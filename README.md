# SNAP-Preprocessing
Sentinel-1 SAR preprocessing using ESA SNAP software.
This project shows the preprocessing workflow for Sentinel-1 SAR data using ESA SNAP software.

## Steps Performed
- Apply Orbit File  
- Remove Thermal Noise  
- Radiometric Calibration  
- Terrain Correction  
- Convert to Sigma0
